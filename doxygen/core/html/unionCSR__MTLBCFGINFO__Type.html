Detailed Description

Union type to access MTLBCFG_INFO CSR register.

NOTE: the MTLBCFG_INFO CSR supports two different mapping layouts. Use the b.mapping or nb.mapping field to determine the active mapping type. If mapping field is set, use the nb structure for field access. Otherwise, use the b structure for field access.
