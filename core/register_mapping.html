The table below associates some common register names used in NMSIS to the register names used in Nuclei ISA Spec.

clicinfo

mth

ECLIC Global Machine Mode Threshold Register

clicintip[i]

ECLIC->CTRL[i].INTIE

clicintie[i]

ECLIC->CTRL[i].INTATTR

mstop
