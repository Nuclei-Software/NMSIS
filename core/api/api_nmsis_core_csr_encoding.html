FRM_RNDMODE_RNE 0x0

FPU Round to Nearest, ties to Even.

FRM_RNDMODE_RTZ 0x1

FPU Round Towards Zero.

FRM_RNDMODE_RDN 0x2

FPU Round Down (towards -inf)

FRM_RNDMODE_RUP 0x3

FPU Round Up (towards +inf)

FRM_RNDMODE_RMM 0x4

FPU Round to nearest, ties to Max Magnitude.

FRM_RNDMODE_DYN 0x7

In instruction’s rm, selects dynamic rounding mode.

In Rounding Mode register, Invalid

FFLAGS_AE_NX (1<<0)

FPU Inexact.

FFLAGS_AE_UF (1<<1)

FPU Underflow.

FFLAGS_AE_OF (1<<2)

FPU Overflow.

FFLAGS_AE_DZ (1<<3)

FPU Divide by Zero.

FFLAGS_AE_NV (1<<4)

FPU Invalid Operation.
