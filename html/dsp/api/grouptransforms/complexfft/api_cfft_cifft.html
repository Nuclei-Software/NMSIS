const q31_t sinTable_q31[FAST_MATH_TABLE_SIZE + 1]

Table values are in Q31 (1.31 fixed-point format) and generation is done in three steps. First, generate sin values in floating point: where PI value is 3.14159265358979

Second, convert floating-point to Q31 (Fixed point): (sinTable[i] * pow(2, 31))

Finally, round to the nearest integer value: sinTable[i] += (sinTable[i] > 0 ? 0.5 : -0.5);

const q15_t sinTable_q15[FAST_MATH_TABLE_SIZE + 1]

Table values are in Q15 (1.15 fixed-point format) and generation is done in three steps. First, generate sin values in floating point: where PI value is 3.14159265358979

Second, convert floating-point to Q15 (Fixed point): (sinTable[i] * pow(2, 15))

Finally, round to the nearest integer value: sinTable[i] += (sinTable[i] > 0 ? 0.5 :-0.5);
