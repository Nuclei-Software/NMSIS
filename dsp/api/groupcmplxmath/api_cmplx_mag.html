group cmplx_mag

Computes the magnitude of the elements of a complex data vector.

The pSrc points to the source data and pDst points to the where the result should be written. numSamples specifies the number of complex samples in the input array and the data is stored in an interleaved fashion (real, imag, real, imag, ‚Ä¶). The input array has a total of 2*numSamples values; the output array has a total of numSamples values.

There are separate functions for floating-point, Q15, and Q31 data types.

RISCV_DSP_ATTRIBUTE void riscv_cmplx_mag_fast_q15 (const q15_t *pSrc, q15_t *pDst, uint32_t numSamples)

Q15 complex magnitude.

Scaling and Overflow Behavior

The function implements 1.15 by 1.15 multiplications and finally output is converted into 2.14 format. Fast functions are less accurate. This function will tend to clamp to 0 the too small values. So sqrt(x*x) = x will not always be true.
